# device.isSimulator

|                      | &nbsp; 
| -------------------- | ---------------------------------------------------------------

## Overview

This property indicates whether the device running the application is the Corona Simulator.


## Example
 
``````lua
local device = require 'plugin.device'

print( device.isSimulator )
``````
